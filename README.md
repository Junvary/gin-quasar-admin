# Gin-Quasar-Admin
gin-vue-admin的前端quasar实现（quasar版本：v1.14.5）

> 鸣谢 https://github.com/flipped-aurora/gin-vue-admin

> 本项目旨在学习 Quasar 使用


*如果你喜欢这个项目，或者对你有一点启发和帮助，请给一个小小的 star， 谢谢！*

#
ElementUI版本：https://github.com/Junvary/gea

Quasar版本：https://github.com/Junvary/gin-quasar-admin

> Quasar版本旨在学习在Vue中使用Quasar Framework，听说ElementUI原班人马已经回归，更要支持了！

> 目前项目的后端初始化方法与 gin-vue-admin 保持一致，但数据库初始化文件内容进行了相应的更改，推荐重新建库体验。初始化前请修改 gea-server 目录下 config.yaml 中 mysql 配置为自己的数据库信息。

> 前端启动： yarn 安装依赖 ， quasar dev 启动前端（代替 yarn run serve）

> 由于 Gin-Quasar-Admin 移除了一些 gin-vue-admin 的功能，如需添加请自行参考原项目。
